需要注意的问题：

  1. 需要保存链表 b 的头指针。头指针对链表的重要性不言而喻，没有头指针则无法找到链表。因此定义一个头指针 LinkNode* head;
  2. 在为链表 b 添加新结点的过程中，需要一个指针变量跟随当前结点。代码结束后这个指针指向链表 b 的最后一个结点，这与尾指针的定义不谋而合，因此不妨定义这个指针为 LinkNode* tail;
  3. 对于链表来说，动态内存分配分为两步：第一步先为 next 指针指向的空间分配内存(p->next = new LinkNode)，第二步再将当前的指针改为 next 指针(p = p->next)。这两步的顺序不可交换，即不可写作(p = p->next; p = new LinkNode)，原因正如上文中的例子，p = new LinkNode 会将一个空内存块的地址赋给指针变量 p，因此 p 会改变，不再是上一个结点的 next 地址。
  4. 必须存在一步：为头指针分配内存，即将头指针指向一块已分配的链表结点内存空间。注意不可写作LinkNode* head = tail; LinkNode* tail = new LinkNode;，这种写法是无效的，这一点与问题3的原因相同。
